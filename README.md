## 개요
node.js 학습을 위해 간단한 대문자, 소문자 알파벳 변환기를 만들어보았습니다.
node.js에 내장되어있는 http 모듈을 이용해 간단히 구현되어있습니다.
화면은 간단하게 반응형으로 구현되어있습니다.

## 툴
- server: `node.js`
- client: `vite`, `typescript`, `react`, `css`

## 시작하기
```bash
npm run client
npm run server
```
